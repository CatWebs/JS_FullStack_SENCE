# **Proyecto: Aplicación de Consola**

Proyecto realizado con JavaScript que responde a las siguientes 5 consignas:

## Archivo 01-leccion-1.js

CONSIGNA DE LA LECCIÓN

- Escribir script que muestre mensajes en consola
- Recibir datos de usuario por consola
- Mostrar mensajes personalizados con alert.

##### Se hacen las actividades solicitadas. No hay validación de datos ni condiciones.

---

## 02-leccion-2.js

CONSIGNA DE LA LECCIÓN

- Definir variables utilizando let y const
- Pedir al usuario que ingrese 2 numeros
- Implementar operaciones matematicas
- Usar estructuras condicionales

##### Sólo se aplican estructuras condicionales. No hay validación de los datos ingresados.

---

## 03-leccion-3.js

CONSIGNA DE LA LECCIÓN

- Crear un arreglo con una lista de elementos
- Usar for y while para recorrer arreglos
- Implementar una función que filtre elementos según una condición

##### Se aplica estructura con For y con do While. Se incluye función de filtro.

---

## 04-leccion-4.js

CONSIGNA DE LA LECCIÓN

- Crear funciones para cada operación matemática
- Implementar función que reciba parámetros y retorne un resultado
- Llamar funciones dentro de otras para optimizar el código

##### Se completa la actividad. No se validan datos

---

## 05-leccion-5.js

CONSIGNA DE LA LECCIÓN

- Crear un objeto con propiedades y valores
- Implementar métodos dentro de un objeto
- Usar un arreglo de objetos y recorrerlo con map() o forEach()

##### Se completa la actividad creando un objeto, accediendo a él (ver en consola) y creando un array de objetos, accediendo a los datos con forEach y con Map (ver en consola)
